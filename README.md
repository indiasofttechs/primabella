# PrimaBella Dental Clinic

Welcome to the **PrimaBella Dental Clinic** project repository! This is a modern, responsive, and aesthetically pleasing landing page for a premium dental clinic located in Discovery Gardens, Dubai. 

## 🦷 Overview

PrimaBella Dental Clinic's website is designed to provide a premium user experience, featuring dynamic designs, dark mode capabilities, and detailed sections about the services offered, the doctors, a gallery, and patient reviews.

## ✨ Features

- **Modern & Premium UI**: Beautiful color palettes, smooth transitions, and high-quality typography using Google Fonts (Inter & Playfair Display).
- **Dark Theme Support**: Built-in toggle for users preferring a darker, sleeker interface.
- **Responsive Layout**: Fully responsive design optimized for all screen sizes (mobile, tablet, desktop).
- **Service Highlights**: Dedicated sections for Cosmetic, General, Implant, Pediatric, and Orthodontic services.
- **Interactive Elements**: Hover effects, floating animations, and a responsive carousel for reviews.
- **Appointment Booking**: Integrated forms for booking appointments directly from the website.

## 🛠️ Technologies Used

- **HTML5**: Semantic markup for better SEO and accessibility.
- **CSS3**: Custom CSS with variables for theming, flexbox/grid for layouts, and smooth animations.
- **JavaScript (Vanilla)**: For interactive elements like the dark mode toggle, review carousel, and modal pop-ups.

## 🚀 Getting Started

To view the project locally:

1. Clone or download the repository.
2. Open the `index.html` file in your preferred modern web browser.
3. No build tools or servers are required for the static version.

## 📁 Project Structure

- `index.html`: The main entry point containing all structure and styling.
- Image assets (e.g., `logo.jpeg`, `hero.png`, service icons, doctor profiles, etc.) used throughout the site.
